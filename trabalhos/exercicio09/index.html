Exemplo de Função JavaScript

<p id="texto">Este é o texto original.</p>

<button onclick="mudarTexto()"></button> 
    <script>
        // Função que muda o texto do parágrafo
        function mudarTexto() {
            document.getElementById('texto').innerText = 'O texto foi alterado!';
        }
    </script>

Eventos mais utilizados

Eventos de Mouse:

click: Disparado quando um elemento é clicado.

dblclick: Disparado quando um elemento é clicado duas vezes rapidamente.

mouseover: Disparado quando o ponteiro do mouse passa sobre um elemento.

mouseout: Disparado quando o ponteiro do mouse sai de um elemento.

Eventos de Teclado:

keydown: Disparado quando uma tecla é pressionada.

keyup: Disparado quando uma tecla é liberada.

keypress: Disparado quando uma tecla é pressionada e liberada.

Eventos de Janela:

load: Disparado quando uma página é carregada.

resize: Disparado quando o tamanho da janela é alterado.

scroll: Disparado quando a página é rolada.

Eventos de Formulário:

submit: Disparado quando um formulário é enviado.

change: Disparado quando o valor de um elemento de formulário é alterado.

focus: Disparado quando um elemento ganha foco.

blur: Disparado quando um elemento perde foco.
